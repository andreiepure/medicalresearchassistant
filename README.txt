Instructions:

1. Add files containing the Medline citations - these will appear in the list.
- filtering will be done while adding - please see 'unique citations'
- when searching for the file, please make sure the correct filter is in place - 'txt' if your files are txt and 'nbib' if your files are nbib

2. Choose the output folder.

3. Choose a value for citations / file - if you want only one resulting file, just add 99999 to the text field.

4. Choose an output name for the file.

5. Press 'Save to files'
- you should see the filtered files in the output folder
